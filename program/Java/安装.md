# Java安装

## Linux安装

- 安装包下载：

~~~
链接：https://pan.baidu.com/s/15PJ7PksMObZajXWaJ9hcMA 
提取码：3x5w
~~~

- 解压安装

~~~
tar -xzvf jdk-8u201-linux-x64.tar.gz
~~~

- 配置环境变量

~~~
vim ~/.bashrc
# 增加以下配置
export JAVA_HOME=/root/software/jdk1.8.0_201
PATH=$PATH:$JAVA_HOME/bin
# 使配置生效
source ~/.bashrc
~~~

- 验证是否成功

~~~
java -version
~~~
